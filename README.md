# Map Share

複数地点を地図上に登録し、共有URLとして渡せる静的Webアプリです。

## できること

- 複数地点の登録
- ピン種類の選択
- 地図上に常時表示されるコメント
- 登録順に道路沿いルートを表示
- 地点データを含む共有URLのコピー

## 使い方

1. 地図をクリック、または場所検索で地点を選びます。
2. 場所名、ピン種類、コメントを入力して地点を追加します。
3. ピンを選択すると表示モードになり、編集ボタンから編集できます。
4. 共有リンクのコピーで、地点データ入りのURLを共有できます。

## 注意

地点データはURLの `#places=` に含まれます。道路沿いルートは公開OSRM APIを使うため、通信状況やAPI側の混雑により表示できない場合があります。
